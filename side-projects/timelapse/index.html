--- layout: page title: "TimeLapse Tasks Manager" --- {% include JB/setup %} TimeLapse Tasks Manager: RIA consisting of a Java EE Back-End ( Struts 2 + flexjson, Hibernate + JPA orchestrated through Spring 2.5 with MySQL as DBMS) and a HTML + JavaScript (Dojo Toolkit 1.3.1) Front-End with JSON for transmitting all data without using any JSP or other view technology on Server. This project was the winner of championship of Final Degree Projects of ETSII of Seville in 2010. The source code of the project is published in this GitHub Repository.
